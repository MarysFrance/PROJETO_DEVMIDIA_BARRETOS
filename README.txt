# 🌆 Barretos

Este repositório contém uma página HTML simples sobre a cidade de **Barretos (SP, Brasil)**, conhecida nacionalmente pelo **Maior Rodeio do Brasil**, por ser referência internacional em **saúde oncológica** e agora também por seu reconhecimento como **estância turística**.

This repository contains a simple HTML page about the city of **Barretos (São Paulo, Brazil)**, nationally known for hosting the **Biggest Rodeo in Brazil**, internationally recognized for its **oncology healthcare excellence**, and recently designated as a **tourist destination**.
